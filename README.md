# Calculator (Tkinter)

A simple GUI calculator built with **Python Tkinter**. Supports basic arithmetic, percentage, sign toggle, decimals, and square root, with a clean button layout and styled interface.

## Features

# Performs without eval function
* Operations: `+  -  ×  ÷  =`
* Functions: `AC`, `+/-`, `%`, `√`
* Decimal support
* Auto-formats whole numbers (removes unnecessary `.0`)
* Prevents leading zeros (e.g., avoids `05`)
